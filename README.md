# David — Public GitHub Pages Profile

This is a privacy-friendly bilingual resume/profile page for GitHub Pages.

## Personal information removed

- Surname removed from page title, hero, navigation and metadata.
- Phone number removed.
- Exact local area replaced with `France / Remote`.
- PDF resume links removed to avoid exposing private information through downloadable files.

## Add your digital twin link

Open `index.html` and replace this value near the bottom of the file:

```js
const DIGITAL_TWIN_URL = "https://YOUR-DIGITAL-TWIN-LINK.com";
```

## Deploy on GitHub Pages

1. Create a GitHub repository.
2. Upload the contents of this folder to the repository root.
3. Go to **Settings → Pages**.
4. Choose **Deploy from branch**.
5. Select `main` and `/root`.
6. Save and wait for GitHub to publish the page.
